# 🌐 Portfólio Multilíngue

Este é o repositório do meu portfólio pessoal, desenvolvido com HTML, CSS e JavaScript puro. Ele apresenta minhas habilidades, projetos e experiências profissionais, com suporte para múltiplos idiomas.

[🔗 Acesse o portfólio online](#)

## ✨ Funcionalidades

- 🌎 Suporte multilíngue (Português, Inglês — e escalável para outros idiomas)
- 🔁 Redirecionamento automático com base no idioma do navegador
- 💼 Sessões sobre mim, habilidades, projetos e participações
- 📱 Design responsivo (em desenvolvimento)
- 💡 Código simples e direto, sem frameworks
